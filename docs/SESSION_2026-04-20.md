### 2. AI 验证理由全是"技术快照构建失败"
- **文件**: `backend/signals/monitor.py:382-389`
- **病因**: `calc_macd()` 返回 `dict {dif, dea, histogram}`，原代码 `dif, dea, hist = calc_macd(closes)` 解包 dict 拿到的是三个字符串（key 名），后续 `hist > 0` 触发 `'>' not supported between str and int`
- **修复**: 按 key 取值 + `float()` 兜底
- **验证**: 快照长度 432 字节，MA/RSI/MACD/ATR/支撑阻力/量能/形态全齐

### 3. 信号永远卡在"⏳ 验证中..."
- **文件**: `backend/signals/monitor.py:498-505`
- **病因**: LLM 调用失败时 `if not verdict: return` 直接走掉，DB 不更新、WS 不推送，前端永远显示验证中
- **修复**: LLM 失败时回填 placeholder verdict（warn / conf=0 / reason="AI 验证未完成..."）并推送
