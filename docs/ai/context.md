# 项目背景与上下文

## 产品定位

OpenChart Pro 是一个面向个人投资者的行情分析工具，核心卖点是将缠论（中国特有的技术分析理论）与现代量化工具结合，提供专业级的图表分析能力。

## 目标用户

- 使用缠论做交易决策的加密货币投资者
- 需要多市场（加密/美股/A股/港股）统一看盘的交易者
- 希望用 AI 辅助分析的个人投资者

## 核心业务逻辑

### 缠论分析

缠论是一套基于 K 线形态的技术分析方法，核心概念：
- **笔**：由顶分型和底分型之间的 K 线构成（至少 5 根）
- **线段**：由至少 3 笔构成
- **中枢**：至少 3 笔重叠形成的价格区间（ZG=上沿, ZD=下沿）
- **买卖点**：一类（趋势背驰）、二类（回调不破前低/高）、三类（中枢突破）

系统使用 chan.py 开源引擎做核心计算，我们只写了适配层（chanlun_service.py）。

### 艾略特波浪

自研模块，识别 5 浪推动 + ABC 修正结构，提供预测目标线。

### 数据源

- 加密货币：OKX（主力）和 Binance 的 REST + WebSocket API
- 美股/港股：Yahoo Finance
- A股：东方财富
- 所有数据通过 `backend/data/` 下的适配器统一接口
